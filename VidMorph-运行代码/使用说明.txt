爆款视频分析工具 - 评委使用说明

1. 双击 VideoTokenTransferApp.exe 启动软件。
2. 本软件推荐在 Windows 系统运行。
3. 软件不需要安装 Python，不需要安装依赖，也不需要打开终端。
4. 软件需要联网调用模型服务。
5. 使用前请在软件内输入火山方舟 API Key。
6. API Key 仅用于本次运行，不会写入源码或交付包文件。

使用流程：

第一步：爆款视频分析
- 在“爆款视频分析”页面输入 API Key。
- 选择或拖入一个爆款视频文件。
- 点击“开始严格 Token 提取”，等待 24 Token 结构分析完成。
- 分析完成后可预览 JSON，并导出 PDF 报告。

第二步：用户素材分析
- 进入“用户素材分析”页面。
- 添加图片、视频或文本素材。
- 点击“开始素材分析”，软件会基于第一步的 24 Token 结果分析素材适配关系。
- 分析完成后可下载素材分析 JSON。

第三步：改进方案生成
- 进入“改进方案生成”页面。
- 确认前两步已完成。
- 点击“生成改进方案”，生成拍同款改进方案。
- 可导出方案报告。

常见情况：

- 如果模型服务繁忙，请稍后重试。
- 如果提示 API Key 错误，请检查 Key 是否正确、是否有可用额度。
- 如果视频过大或格式不支持，请换用常见格式视频，例如 MP4、MOV、AVI、MKV、WEBM。
